Требования по сборке:
CMake версии 3.16 или выше
Компилятор GCC версии 5.3.1 или выше.

Инструкция по сборке:
mkdir build
cd build
cmake ..
make

Параметры запуска
•-d image_dir – Параметр для указания директории для сохранения изображений
•-h – Опция справки
•-i input_file – Параметр для указания входного файла
•-l – Параметр для сохранения гиперссылок
•-m – Параметр для вставки метаданных документа
•-r – Параметр для сохранения таблиц в сыром виде (без переносов и установки символов разделения `|` и `-`)
•-n – Параметр для сохранения заметок
•-o output_file – Параметр для указания директории для сохранения выходного файла
